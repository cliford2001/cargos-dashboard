# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Logo SVG integrado en header con animación de carga
- Micro-interacciones suaves (hover, transiciones)
- Mejora de tipografía y jerarquía visual con Inter

## [0.1.0] - 2026-07-09

### Added
- Primera versión del dashboard de finanzas personales
- Carga dinámica de movimientos desde data/cargos.json
- Cálculo automático de entradas, salidas y balance
- Soporte para periodo salarial desde último sueldo

### Changed
- Estilo visual inicial con paleta oscura/clara
- Diseño responsive para móviles y desktop

### Fixed
- Corrección de orden de movimientos por fecha/hora
- Manejo de esquema antiguo (cargos en lugar de movimientos)

## [0.0.1] - 2026-07-09

### Added
- Repositorio inicial con index.html y data/cargos.json de ejemplo
